# Tópicos Importantes

- Qual a linguagem de programação utilizada na disciplina? JavaScript

- O que é o Node? ambiente que executa o js, no terminal ou no servidor 

- O que é Framework e qual a diferença para uma biblioteca? o framework é um conjunto de recuros ou feramentas para te ajudar a resolver um problema. framework orienta melhor para o seu uso ja a biblioteca não

- O que é o NPM? node package menager, gerenciador de pacotes do node, adicionar pacotes, rodar scripts e remover pacotes.

- O que é React? é uma biblioteca js, para criar uma interface, baseada em componentes que permite organizar e facilitar a manutenção de um codigo front-end "escalabilidade" do codigo

- O que é o React Native? é uma biblioteca que permite criar aplicativos mobile em js, adiciona ao react os recursos necessarios pra desenvolvimento mobal

- O que é o Expo? framework do reactNative.

-Qual é a diferença entre as Libs react, ract native, react dom e expo? React= base Native=complemento pra mobile Dom=complemento pra web Expo=Framework mobile.

- Como estilizar um componente no React Native? é inpirado no css e vc usa {Styles = stylesheet.create} é quase igual ao css

- O estilo CSS in JS no React Native é igual ao CSS Web? Tenho todas as propriedades?Não, Não tem propriedades especificas

- Em quais pacotes encontro um componente de Imagem? Qual melhor para usarmos atualmente?React Native e Expo 

-O que são componentes do React?é um pedaço da interface 

- O que são propriedades de componentes? sao dados que podemos alterar dos componentes no momento em que chamamos ou é criado 

- O que são Hooks? são funções do react q nem no html vem antes do return pricipais {useState, useEffect}. Usados só no componente, todos os Hooks começam com Use

-Para que serve e como funciona o useState? é um hook para criar estados e funções que alteram o estado, sempre que a função altera o estado a tela é atulaizada com o novo valor.   

-Para que serve o useEffect? é um hook que funciona como gatilho e toda vez que um dos estados é alterado que ela ta observando muda ela dispara uma função
